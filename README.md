# LuckyDog
一个简单的点名器（无后续开发计划）

## 构建
1. 安装cmake和Qt6
2. 克隆此仓库
    ```bash
    git clone https://github.com/constant-e/LuckyDog.git
    ```
3. 创建`build`文件夹
    ```bash
    mkdir LuckyDog/build && cd LuckyDog/build
    ```
4. 运行`cmake`
   ```bash
   cmake ..
   ```
5. 运行`make`
    ```bash
    make
    ```
